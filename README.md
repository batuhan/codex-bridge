# Codex Bridge

Codex Bridge lets you put Codex into Beeper.

Beeper messages are end-to-end encrypted, so we do not see your messages.

## Run

You need Go and the Codex CLI installed.

```sh
./run.sh
```

The script logs you into Beeper and Codex if needed, generates `config.yaml`, builds the bridge, and starts it.
